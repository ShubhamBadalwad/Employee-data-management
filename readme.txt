						*** Employee Data Management System ***



Employee data management program will manage the basic data for the employees.
In this program we have to first login by giving username and password. Then username and password are verified against the authenticated
username and password. It also remembers the Log of activities. Employee Data Management System has following modules

Login: to run this program we have to login first. The authenticated username and password is in paswrd.txt

Features to be implemented in future:-
1) we will be able to change the password.
2) We will be able to keep log of activities.
3) We will be able to see the deleted entries in "trash".
4) We will be able to restore the deleted entries from "trash".
